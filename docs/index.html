BigCommerce WYSIWYG Editor

A free, open-source Chrome extension that adds a modern, native WYSIWYG editor to BigCommerce Page Builder. Effortless rich text editing, no external dependencies, and full privacy.

Install from Chrome Web Store View on GitHub

Features

  • Automatic detection in BigCommerce Page Builder
  • Native WYSIWYG editor for HTML fields
  • Full text formatting: bold, italic, underline, lists, alignment
  • Advanced tools: tables, images, color selection
  • Real-time updates as you edit
  • Efficient resource management
  • Modern, intuitive interface
  • No external library dependencies
  • Chrome extension security policy compliant
  • Completely free and open-source

How it works

  1. Log in to your BigCommerce admin panel
  2. Go to Page Builder
  3. Edit an HTML text field – the WYSIWYG editor appears automatically
  4. Format your content visually and save
