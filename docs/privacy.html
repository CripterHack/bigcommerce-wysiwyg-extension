Privacy Policy

Last updated: 21 July 2025

1. Introduction

BigCommerce WYSIWYG Editor (the "Extension") is a Chrome browser extension that adds a rich‑text toolbar to HTML blocks inside BigCommerce Page Builder. We respect your privacy and are committed to protecting it. This document explains what information we collect, how we use it, and what choices you have. In short: we do not collect, store, or transmit any personal data.

2. Data We Collect

We do not:

  • Collect personally identifiable information (PII)
  • Track browsing history or user activity outside BigCommerce Page Builder
  • Store or transmit the content you edit in your store

The Extension operates entirely in your local browser context and never sends data to external servers.

3. Local Storage

The Extension may save a few non‑personal settings (e.g., whether the toolbar starts minimized) using Chrome's storage API. These values are:

  • Stored only on your device
  • Not synced to our servers or anyone else's
  • Removable at any time via the Chrome "Clear browsing data → Site settings → Hosted app data" dialog or by uninstalling the Extension

4. Chrome Permissions Explained

Permission Why It's Needed Scope
activeTab Allows the Extension to run its script only after you interact with it (click icon) in the current BigCommerce tab Single active tab
storage Saves local UI preferences (no user data) Your browser profile
tabs Reads the current tab's URL to confirm it matches *.mybigcommerce.com before activation Tab URL only
Host *://*.mybigcommerce.com/* Restricts execution exclusively to BigCommerce domains No other sites

5. Cookies and Tracking

The Extension sets no cookies and implements no tracking or analytics mechanisms.

6. Third‑Party Data Sharing

We do not share data with third parties, affiliates, advertisers, or analytics providers because no data is collected in the first place.

7. Security

All code is bundled inside the Extension package and reviewed publicly on GitHub. No remote code is downloaded or executed at runtime, minimizing supply‑chain risks. Updates are delivered only through the official Chrome Web Store.

8. Children's Privacy

The Extension is intended for users aged 18 and above who manage BigCommerce stores. It collects no data from anyone, including children under 13.

9. Changes to This Policy

If future updates introduce features that require data collection, we will:

  1. Update this policy and the Extension's listing before release
  2. Obtain explicit user consent where applicable
  3. Limit data collection to what is strictly necessary for new features

10. Contact

If you have questions about this Privacy Policy, please email dev@izignamx.com or open an issue in the GitHub repository.



By installing or using the Extension you acknowledge that you have read and understood this Privacy Policy.
